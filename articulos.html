¿Por que estructurar correctamente una página HTML?

Estructurar una página HTML de forma correcta es fundamental para que el sitio sea fácil de leer, mantener y entender, tanto para las personas como para los motores de búsqueda.

Bases en HTML

Comiencé con lo más básico: crear un archivo .html y escribir las etiquetas <head>,<body>,etc. Después implemente encabezados con <h1>, <h6>,<p> y enlaces con <a>.
